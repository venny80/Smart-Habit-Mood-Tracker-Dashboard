# Smart Habit + Mood Tracker


Candidate: Matta Vennela


## What is this
A small, human-friendly web app to track daily habits and mood entries. It includes:
- Habits CRUD (create/edit/delete)
- Mood entries CRUD (date, mood, note)
- Simple charts for mood trends and habit completion
- Mock mode for frontend-only submissions


This project is built to be clear, authentic, and easy to run locally.


## Tech stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: SQLite (file-based)
- Charts: Chart.js


## How to run (local)
